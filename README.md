# Customer-Support-Performance-Analysis
"Excel Dashboard to analyze customer support KPIs like CSAT and Resolution Time."
Project Overview
This project focuses on analyzing customer support performance using a dataset of support tickets. The goal was to transform raw data into actionable business insights to improve agent efficiency and customer satisfaction (CSAT).

Tools Used
Microsoft Excel: Data Cleaning, Pivot Tables, and Dashboarding.

Data Visualization: Clustered Column Charts, Pie Charts, and Interactive Slicers.

 Key Business Insights
After analyzing the data, I discovered several critical performance metrics:

Top Performance: * Agent Sneha achieved the highest Average CSAT score of 4.67, setting a benchmark for the team.

Process Bottlenecks: * The Billing category has the highest average resolution time (14.5 hours), indicating a need for better documentation or training in this area.

SLA Compliance: * Around 67% of tickets are breaching the 10-hour resolution target. This highlights a significant gap in meeting Service Level Agreements (SLA).

Improvement Areas: * Agent Priya requires additional support or training to improve her CSAT scores and align with the team average.
